# 🍫 ChocoAnalytics: Inteligência de Vendas & Eficiência de Marketing

Este repositório contém uma análise de dados focada no comportamento do mercado de chocolates em três canais de distribuição: **Online, Varejo e Atacado**. O projeto foi prototipado no **Marimo** e transformado em um aplicativo interativo usando **Streamlit**.

O objetivo é responder a duas perguntas estratégicas de negócio:
1. Os descontos aplicados estão realmente trazendo mais volume ou apenas queimando margem?
2. Quais produtos dão o melhor retorno para cada real investido em marketing?

---

## 📊 Principais Insights

* **Descontos vs. Volume (`Boxes_Shipped`):** Identificamos que a correlação entre a taxa de desconto e o volume de caixas de chocolate despachadas é fraca nos canais principais. Isso indica margem para reduzir descontos sem derrubar o volume.
* **Campeões de ROI (`Marketing_Spend`):** O ranking de eficiência isolou os chocolates que "se vendem sozinhos", apresentando o maior faturamento proporcional ao custo de anúncio.
* **Evolução Temporal:** Gráficos de linha acompanham o faturamento histórico de cada canal para identificar sazonalidades (como períodos de Páscoa ou datas comemorativas).
